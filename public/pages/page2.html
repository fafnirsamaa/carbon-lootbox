Hypothèses de calcul

Catégories d'émissions

Les données du Bilan Carbone N-1 de Niji sont utilisées dans cet outil pour tous les postes d'émissions du niveau 0, et pour le poste "Site Niji" des niveaux 1 & 2. Un ratio est a été appliqué sur ces données, afin d'obtenir les émissions par ETP et par jour ouvré.

Autres (Site Niji)

Les émissions de ce poste regroupent les catégories Achats de bien, Achats de service, Déchets, Immobilisations autres équipements / véhicules / bâtiment, Télétravail. Celles-ci figurent dans le Bilan Carbone Niji N-1, et sont ramenées à 1 ETP/jour ouvré.

Fuites de fluide de climatisation

3 hypothèses sont déclinables selon le niveau d'information disponibles pour les équipements de clim sur les sites. Lorsque la donnée de fuite de fluide de climatisation d'un site n'est pas renseignée dans le Bilan Carbone N-1 :

La quantité de fluide dépend de la surface occupée (équivalent à 0,1 kW frigorifique par m²)

La quantité totale de fluide dans le système de climatisation est de 0,3 kg par kW frigorifique, auquel un taux de fuite annuel moyen de 10% est appliqué.

Le R410A est choisi pour les climatiseurs dont le fluide frigorigène est inconnu, car il s'agit du fluide frigorigène le plus utilisé dans les climatisations tertiaires.

Consommation d'énergie

Lorsque la donnée de consommation énergétique d'un site n'est pas renseignée dans le Bilan Carbone N-1, mais disponible l'année précédente :

Hypothèse d'une consommation identique à l'année précédente lorsqu'il n'y a pas eu d'augmentation de l'effectif.

Lorsqu'il n'existe pas de données de consommations énergétiques pour un site :

Hypothèse d'une consommation électrique annuelle de 112 kWh/m² par an (source OID).

Infrastructure numérique

Hypothèse d'une durée de vie d'un équipement identique à la durée d'amortissement de chaque équipement Niji.
